# Dragon's Hoard Summary:
     This project was my final project for CS50: Introduction to Computer Science course. The project was about making a video game, with the goal to showcase what I've learned throughtout the course, as well as what I've learned independently. I created an arcade style game where the user controlled a dragon and shot fireballs to destroy oncoming knights that were trying to steal gold from your hoard.
     In this project I learned how to code using C#, how to use Godot engine, how to use API, and general concepts of object oriented programming. I was the sole owner of the project and was responsbile for all aspects of the project. My experience creating this game provided me with a strong foundation on how to manage my own projects, independently learn, apply my knowledge from my course work, and how to have fun coding.
